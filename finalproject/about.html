About FitForge

FitForge is a teaching project demonstrating a responsive site using semantic HTML, CSS, and modern JavaScript. This site includes data fetching, dynamic DOM generation, localStorage persistence, and accessible modals.

Design Principles

We used proximity, alignment, repetition, contrast, and whitespace to provide a consistent, readable UI at mobile and desktop breakpoints.
