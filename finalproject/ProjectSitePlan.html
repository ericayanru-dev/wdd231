Project by Eric Ayanru

Site Name

Beginner Code Path

This name reflects the purpose of the website, which is to provide beginner-friendly coding resources, tutorials, and examples for those starting their web development journey.

Optional Domain: beginnercodepath.org

Site Purpose

The site provides a hub of learning resources, projects, and tutorials for beginner web developers. Visitors will find structured learning paths, sample projects, and practical exercises to help them build confidence in HTML, CSS, and JavaScript.

Scenarios

  • “Where can I find beginner-friendly coding tutorials in HTML and CSS?”
  • “What projects can I build to practice and showcase my skills as a beginner?”

Color Schema

Selected Colors:

  • #1E90FF (Dodger Blue) – Used for headings and accents.
  • #333333 (Dark Gray) – Used for body text.
  • #F4F4F4 (Light Gray) – Used as background color.

Typography

Selected Fonts:

  • Roboto – Used for body text.
  • Poppins – Used for headings.
